  - `npm start` start the server.

#### 2. HapiJS - [README](https://github.com/JSbotHQ/RealTime-Faye/tree/master/HapiJS)

  - `npm start` start the server

#### 3. TrailsJS - [README](https://github.com/JSbotHQ/RealTime-Faye/tree/master/TrailsJS)
  - `cd TrailsJS`
  - `npm start` start the server
